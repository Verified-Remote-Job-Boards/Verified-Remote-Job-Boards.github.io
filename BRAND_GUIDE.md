# Verified Remote Job Boards - Brand Style Guide

## Logo
- **Primary Logo**: `/images/logo.png` (horizontal layout with shield + text)
- **Icon Logo**: `/images/logo-icon.png` (shield with checkmark only, for favicon/social)

## Color Palette

### Primary Colors
- **Deep Blue**: `#1e40af` (rgb: 30, 64, 175)
  - Use for: Logo, primary buttons, headings, links
  - Conveys: Trust, authority, professionalism

- **Dark Gray**: `#374151` (rgb: 55, 65, 81)
  - Use for: Body text, secondary headings
  - Conveys: Professionalism, readability

### Secondary Colors
- **Light Blue**: `#3b82f6` (rgb: 59, 130, 246)
  - Use for: Hover states, accents, highlights
  
- **Success Green**: `#10b981` (rgb: 16, 185, 129)
  - Use for: Positive reviews, success messages, "verified" badges

- **Warning Orange**: `#f59e0b` (rgb: 245, 158, 11)
  - Use for: Mixed reviews, caution notices

- **Neutral Red**: `#ef4444` (rgb: 239, 68, 68)
  - Use for: Negative aspects in reviews (use sparingly)

### Background Colors
- **White**: `#ffffff`
  - Primary background
  
- **Light Gray**: `#f9fafb` (rgb: 249, 250, 251)
  - Alternate sections, cards, table rows

- **Border Gray**: `#e5e7eb` (rgb: 229, 231, 235)
  - Borders, dividers

## Typography

### Recommended Font Stack
```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Helvetica Neue', Arial, sans-serif;
```

### Font Sizes
- **H1 (Page Title)**: 36px, bold, #1e40af
- **H2 (Section Heading)**: 28px, bold, #1e40af
- **H3 (Subsection)**: 22px, semibold, #374151
- **Body Text**: 16px, regular, #374151
- **Small Text**: 14px, regular, #6b7280

### Line Height
- Headings: 1.2
- Body text: 1.6

## Design Principles

### Visual Identity
- **Clean & Professional**: Minimal clutter, plenty of white space
- **Trustworthy**: Use shield icon consistently to reinforce verification theme
- **Accessible**: High contrast ratios, readable fonts
- **Modern**: Contemporary design without being trendy

### Content Tone
- **Authoritative**: Position as the definitive resource
- **Helpful**: Focus on user value and practical information
- **Objective**: Balanced reviews with pros and cons
- **Current**: Always include "Last Updated" timestamps

## UI Components

### Buttons
- **Primary**: Deep blue background (#1e40af), white text, rounded corners
- **Secondary**: White background, deep blue border and text
- **Hover**: Light blue (#3b82f6)

### Cards
- White background, light gray border, subtle shadow
- Rounded corners (8px)
- Padding: 24px

### Tables
- Alternating row colors (white / light gray)
- Bold headers in deep blue
- Clear borders in border gray

### Badges
- "Verified" badge: Success green background, white text
- "Premium" badge: Deep blue background, white text
- "Free" badge: Light gray background, dark gray text

## Favicon
Use `/images/logo-icon.png` converted to `.ico` format for browser favicon.

## Social Media
- Use logo-icon.png for profile pictures
- Use full logo.png for cover images and posts
- Maintain consistent color scheme across all platforms
